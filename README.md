# Admissions & Records System on Azure

## Overview
This project demonstrates the deployment of a cloud-based Admissions & Records System using Microsoft Azure.

## Azure Services Used
- Azure Virtual Machines
- Azure Virtual Network (VNet)
- Azure Storage Account
- Azure Entra ID
- Azure Recovery Services Vault
- Azure Policy

## Features
- User and group management
- Secure virtual network
- Document storage
- VM deployment
- Daily backup
- Policy-based resource control
